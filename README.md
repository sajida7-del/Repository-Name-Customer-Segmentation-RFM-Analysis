# Customer Segmentation using RFM Analysis

## Project Overview
This project performs customer segmentation using RFM (Recency, Frequency, Monetary) analysis.

## Tools Used
- Python
- Pandas
- Google Colab
- MySQL
- SQL Optimization

## Steps Performed
- Data Cleaning
- RFM Analysis
- Customer Segmentation
- Exported analytical datasets
- SQL indexing for performance improvement

## Files
- MainData.csv → Cleaned dataset
- AddAnlys.csv → RFM analysis output
- Notebook → Complete workflow

## SQL Work
- Data imported into MySQL
- Created index on CUSTOMER_ID
- Used EXPLAIN for query optimization

## Conclusion
Customer behavior analyzed successfully and optimized database queries improved performance.
